# 📄 AI Resume Screening System

## 🚀 Overview
This project uses Machine Learning to predict job roles from resumes and assign an AI-based resume score.

---

## ✨ Features
- 🔐 Admin Login System
- 📄 PDF Resume Upload
- 🧠 ML-based Job Role Prediction
- 📊 AI Resume Scoring (0–100)
- ⚡ Streamlit Web UI
- 📦 Clean Modular Code

---

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- PyPDF2

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
python train_model.py
python -m streamlit run app.py
